Una API (Interfaz de Programación de Aplicaciones, por sus siglas en inglés) es un conjunto de reglas y protocolos que permite 1  que diferentes aplicaciones de software se comuniquen e intercambien datos entre sí. 2  En términos sencillos, una API actúa como un intermediario que facilita la interacción entre dos sistemas, permitiéndoles compartir información y funcionalidades de manera eficiente.
